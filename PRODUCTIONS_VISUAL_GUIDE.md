## 7. Filter Flow

### Scenario: Find all "Shooting" productions in Mumbai

```
1. User lands on /productions
   ↓
2. Clicks filter button "🎥 Shooting"
   ↓
3. Grid updates to show only shooting projects
   ↓
4. User types "Mumbai" in location search
   ↓
5. Results filter to: Shooting + Mumbai location
   ↓
6. Shows only matching productions
```

### Result
```
Before:  10 cards (mixed statuses, all locations)
After:   3 cards (only shooting, only Mumbai)
```

## 8. Status Transition Timeline

### Example: From Casting to Completed

```
Day 1:  Create Production → Status: CASTING
        [🎬 Casting] badge appears
        Announcement: "Casting open!"
        
Day 15: Update Status → PRE-PRODUCTION
        [📋 Pre-Production] badge updates
        Announcement: "Casting closed, prep phase"
        
Day 30: Update Status → SHOOTING
        [🎥 Shooting] badge updates
        Announcement: "Filming started!"
        
Day 45: Update Status → POST-PRODUCTION
        [🎞️ Post-Production] badge updates
        Announcement: "In post, wrap coming soon"
        
Day 60: Update Status → COMPLETED
        [✓ Completed] badge updates
        Announcement: "Premiere coming soon!"
```

```

(Purple to darker purple)
```
